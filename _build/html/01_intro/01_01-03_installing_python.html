1.2. Εγκατάσταση της Python#

Το πιο δύσκολο πράγμα για την εργασία με την Python (ή οποιαδήποτε γλώσσα προγραμματισμού) είναι να μάθετε πώς να την εγκαταστήσετε σωστά. Γιατί είναι αυτό τόσο δύσκολο; Επειδή η εγκατάσταση της Python διαφέρει δραματικά ανά λειτουργικό σύστημα και έκδοση λειτουργικού συστήματος. Παρακάτω βλέπετε πέντε διαφορετικούς τρόπους με τους οποίους μπορείτε να ξεκινήσετε προγραμματισμό Python από τον ευκολότερο στον πιο δύσκολο.

  1. Χρήση των εφαρμογών Trinket που είναι ενσωματωμένες σε όλο αυτό το εγχειρίδιο.

  2. Χρήση του Google Colab για κάθε ξεχωριστή ενότητα του εγχειριδίου

  3. Χρήση της δωρεάν ενσωματωμένης δυνατότητας Binder αυτού του εγχειριδίου

  4. Χρήση ενός δωρεάν σημειωματάριου Jupyter Notebook στο διαδίκτυο

  5. Εγκατάσταση της Python τοπικά μέσω του Anaconda Navigator και χρήση του Jupyter-Lab

Εάν είστε εντελώς νέοι στον προγραμματισμό, συνιστώ να αγνοήσετε τις επιλογές 2-4 και απλώς να χρησιμοποιήσετε την επιλογή 1 κατά το πρώτο μέρος αυτού του εγχειριδίου. Στην εμπειρία μου, είναι σημαντικό στις αρχές του προγραμματιστικού ταξιδιού σας να έχετε όσο το δυνατόν λιγότερα εμπόδια. Εάν ενδιαφέρεστε να μπείτε στην Python όσο το δυνατόν γρήγορα, τότε χρησιμοποιήστε τα ελεύθερα διαθέσιμα γραφικά στοιχεία Trinket στην ηλεκτρονική έκδοση αυτού του εγχειριδίου και επιστρέψτε στην εγκατάσταση της Python αργότερα.

1.2.1. Trinket#

Σε όλο αυτό το εγχειρίδιο, θα δείτε εφαρμογές Trinket ενσωματωμένες σε κάθε σελίδα. Το Trinket σας επιτρέπει να εξασκήσετε τις δεξιότητες κωδικοποίησής σας χωρίς να εγκαταστήσετε ποτέ την Python. Εάν είστε νέοι στον προγραμματισμό και θέλετε να ξεκινήσετε αμέσως χωρίς να εγκαταστήσετε την Python, συνιστώ να εργαστείτε με το Trinket μέσα στην ψηφιακή έκδοση αυτού του εγχειριδίου.

Αυτά θα σας επιτρέψουν να εξασκηθείτε στο μάθημα ακριβώς μέσα στο ψηφιακό εγχειρίδιο. Οι εφαρμογές Trinket μοιάζουν ως εξής:

from IPython.display import IFrame
IFrame('https://trinket.io/embed/python3/3fe4c8f3f4', 700, 500)

Θα μπορείτε να γράψετε τον κώδικά σας μέσα σε αυτήν την εφαρμογή Trinket και στη συνέχεια να κάνετε κλικ στο κουμπί αναπαραγωγής στο επάνω μέρος. Αυτό θα εκτελέσει τον κώδικά σας και θα συμπληρώσει τα αποτελέσματα στη δεξιά έξοδο.

1.2.2. Χρήση του Google Colab#

Κατά το πρώτο μέρος του εγχειριδίου, σχεδόν όλος ο κώδικας μπορεί να εκτελεστεί με το Trinket. Ακόμη και η ενότητα για το Pandas μπορεί να γίνει μέσα στο Trinket. Ωστόσο, οι μεταγενέστερες ενότητες θα γίνουν ολοένα δυσκολότερες καθώς απαιτούν βιβλιοθήκες που δεν υποστηρίζονται από το Trinket. Σε αυτό το σημείο, θα θέλετε να εξετάσετε άλλη επιλογή. Πάλι, για να καθυστερήσετε την εγκατάσταση της Python στο μηχάνημά σας, μπορούμε να χρησιμοποιήσουμε άλλες λύσεις βασισμένες στο νέφος.

Υπάρχουν δύο επιλογές για την εκτέλεση του κώδικα αυτού του εγχειριδίου στο νέφος έξω από το εγχειρίδιο: Google Colab και Binder (αναφέρεται παρακάτω). Στην επάνω δεξιά γωνία κάθε σελίδας υπάρχει ένας πύραυλος. Εάν τοποθετήσετε το ποντίκι σας πάνω από αυτόν τον πύραυλο, θα έχετε δύο επιλογές για να επιλέξετε από, Binder ή Colab. Κάντε κλικ Colab.

Μόλις κάνετε κλικ στο κουμπί Colab θα μεταφερθείτε στο Google Colab και θα δείτε ένα Notebook μέσα στο σύστημα Google Colab που θα πρέπει να μοιάζει ως εξής:

Όταν είστε εδώ, μπορείτε να εκτελέσετε το notebook.

Το Colab και το Binder έχουν τα πλεονεκτήματά τους. Το Google Colab θα είναι πολύ πιο γρήγορο, αλλά θα απαιτήσει να εγκαταστήσετε τις απαιτούμενες βιβλιοθήκες για κάθε σελίδα. Για το μεγαλύτερο μέρος του πρώτου μέρους του εγχειριδίου, αυτό δεν θα είναι απαραίτητο καθώς δεν θα δουλεύουμε με βιβλιοθήκες. Θα μάθετε πώς να εγκαθιστάνετε βιβλιοθήκες τοπικά και σε ένα Jupyter Notebook (όπως αυτά που χρησιμοποιούνται στο Google Colab) αργότερα σε αυτό το μέρος του εγχειριδίου. Μόλις εγκατασταθούν οι βιβλιοθήκες, ολόκληρο το notebook μπορεί να εκτελεστεί ως εάν το εκτελούσατε τοπικά. Ένα μειονέκτημα του Google Colab είναι η αδυναμία του να αξιοποιήσει τα δεδομένα που βρίσκονται σε αυτό το εγχειρίδιο. Αυτό σημαίνει ότι σε ορισμένες περιπτώσεις, θα πρέπει να ανεβάσετε ручноύ τα δεδομένα στο Google Colab για να εκτελεστούν ορισμένα notebooks όπως αναμένεται.

Ένα άλλο όφελος του Google Colab είναι ότι είναι συνδεδεμένο με τον λογαριασμό σας στο Google. Αυτό σημαίνει ότι οποιαδήποτε αλλαγή που κάνετε στο notebook μπορεί να αποθηκευτεί στο Google Drive σας και να χρησιμοποιηθεί αργότερα. Κάθε φορά που ανοίγετε το notebook, ωστόσο, δημιουργείται ένα μοναδικό παράδειγμα Python, γεγονός που σημαίνει ότι θα πρέπει να εγκαταστήσετε ξανά τις βιβλιοθήκες σας. Ευτυχώς, αυτή η διαδικασία είναι αρκετά γρήγορη.

1.2.3. Χρήση του Binder από αυτό το JupyterBook#

Μια άλλη λύση βασισμένη στο νέφος είναι με το Binder. Σε αντίθεση με το Google Colab, το Binder θα φορτώσει ένα περιβάλλον για ολόκληρο το εγχειρίδιο και θα εγκαταστήσει όλες τις βιβλιοθήκες χρησιμοποιώντας το αρχείο requirements.txt που βρίσκεται στο αποθετήριο GitHub αυτού του εγχειριδίου. Αυτό σημαίνει ότι το Binder θα χρειαστεί σημαντικά περισσότερο χρόνο για να φορτώσει, αλλά θα περιέχει ολόκληρο το εγχειρίδιο. Θα αποθηκεύσει επίσης το παράδειγμα για μια χρονική περίοδο ώστε εάν επιστρέψετε στον ίδιο σύνδεσμο σε σύντομο χρονικό διάστημα, υπάρχει πιθανότητα να μην χρειαστεί να εγκαταστήσει ξανά όλες τις βιβλιοθήκες. Στην εμπειρία μου, το Binder μπορεί να αντιμετωπίσει ορισμένα προβλήματα που μπορεί να είναι δύσκολο να διαγνωστούν, ιδίως εάν είστε νέοι στον προγραμματισμό. Για αυτόν τον λόγο, θα συνιστούσα το Google Colab έναντι του Binder σε νέους προγραμματιστές.

Όπως και με το Google Colab, για να ξεκινήσετε, τοποθετήστε το ποντίκι σας πάνω από το εικονίδιο του πυράλου στο επάνω-αριστερό τμήμα της σελίδας.

1.2.4. Χρήση Jupyter Notebooks Online#

Εάν θέλετε να είστε λίγο πιο ανεξάρτητοι και να μάθετε πώς να κωδικοποιείτε στην Python στο διαδίκτυο χωρίς το Binder και αυτό το εγχειρίδιο, μπορείτε! Δεν χρειάζεται να κατεβάσετε και να εγκαταστήσετε την Python στον τοπικό σας υπολογιστή. Αντίθετα, μπορείτε να χρησιμοποιήσετε δωρεάν σε σύνδεσμο διαδικτύου compilers που σας επιτρέπουν να ακολουθήσετε σχεδόν ολόκληρο αυτό το εγχειρίδιο χωρίς πρόβλημα.

Εάν αυτή είναι η άποψή σας αυτήν τη στιγμή, τότε κάντε κλικ στον σύνδεσμο: https://jupyter.org/try

1.2.5. Εγκατάσταση της Python τοπικά#

Εάν επιθυμείτε να εγκαταστήσετε την Python τοπικά και είναι η πρώτη φορά, υπάρχουν πολλά προβλήματα που μπορούν να προκύψουν. Για αυτόν τον λόγο, τώρα συνιστώ σε όλους τους μαθητές να εγκαταστήσουν μέσω του Anaconda Navigator. Προσθέτει επιπλέον βήματα στη διαδικασία εγκατάστασης, αλλά εξαλείφει το δυνατό για λάθη που θα προκύψουν.

Το Anaconda Navigator είναι μια φιλική προς τον χρήστη διεπαφή που χειρίζεται τη διαδικασία εγκατάστασης για σας. Επιτρέπει επίσης να δημιουργήσετε περιβάλλοντα, τα οποία είναι μικρές περιοχές στον υπολογιστή σας που έχουν μια μοναδική έκδοση της Python και βιβλιοθηκών που είναι εγκατεστημένες. Θα μάθουμε περισσότερα σχετικά με αυτό στο Μέρος 06 όταν εξερευνούμε τις βιβλιοθήκες.

Σε αυτό το κεφάλαιο, θα σας καθοδηγήσω στα βήματα εγκατάστασης του Anaconda Navigator στο μηχάνημά σας, ανεξάρτητα από το λειτουργικό σύστημα.

Όταν γράφουμε κώδικα στην Python, το κάνουμε με λίγους διαφορετικούς τρόπους, ανάλογα με τη χρήση αυτού του κώδικα. Επειδή αυτό είναι ένα εγχειρίδιο και ο κώδικας που γράφω είναι για σκοπούς παρουσίασης, χρησιμοποιώ ένα Jupyter Notebook. Άλλες φορές, ενδέχεται να γράψετε ένα πρόγραμμα σε ένα IDE, ή ένα Integrated Development Environment. Ορισμένα από αυτά περιλαμβάνουν PyCharm, Atom, κ.λπ. Σε άλλες περιπτώσεις, θα χρησιμοποιήσετε την Python στο τερματικό για να εκτελέσετε γρήγορες εργασίες σε δεδομένα σε έναν κατάλογο.

Το Anaconda Navigator αφαιρεί την ανάγκη για σας να μάθετε πώς να το κάνετε όλο αυτό επειδή σας επιτρέπει να εγκαταστήσετε εύκολα το Jupyter-Lab το οποίο λειτουργεί σαν IDE αλλά είναι λίγο πιο συγχωρητικό. Επιπλέον, μπορείτε να καλέσετε περιόδους τερματικού. Ξέρω ότι αυτοί οι όροι δεν έχουν νόημα αυτήν τη στιγμή, αλλά καθώς η κατανόησή σας του προγραμματισμού επεκταθεί, αυτή η παράγραφος θα έχει περισσότερο νόημα. Προς το παρόν, απλώς κατανοήστε ότι το Anaconda Navigator και το Jupyter-Lab (και τα δύο των οποίων εγκαθιστούμε σε αυτό το κεφάλαιο), κάνουν την έναρξη της μάθησης προγραμματισμού πολύ, πολύ απλούστερη.

1.2.5.1. Λήψη του Anaconda Navigator#

Για να κατεβάσετε το Anaconda Navigator, πηγαίνετε στον σύνδεσμο αυτό: https://www.anaconda.com/. Στη συνέχεια, τοποθετήστε το ποντίκι σας πάνω από τα Προϊόντα και επιλέξτε Ατομική Έκδοση, όπως φαίνεται στην παρακάτω εικόνα.

Τώρα, κάντε κλικ στο κουμπί Download όπως φαίνεται στην παρακάτω εικόνα και περιηγηθείτε στη διαδικασία εγκατάστασης στον υπολογιστή σας. Μόλις ολοκληρωθεί, θα έχετε το Anaconda Navigator εγκατεστημένο.

1.2.5.2. Χρήση του Anaconda Navigator#

Τώρα που έχει εγκατασταθεί το Anaconda Navigator, μπορείτε να το χρησιμοποιήσετε ανοίγοντάς το στον υπολογιστή σας. Όταν πρώτη φορά δείτε το Anaconda Navigator, θα δείτε αρκετές επιλογές στα αριστερά. Στα περιβάλλοντα, θα δείτε μόνο ένα περιβάλλον: base. Μην ανησυχείτε για περιβάλλοντα σε αυτό το πρώιμο στάδιο της καριέρας σας στην Python. Προς το παρόν, θα κάνουμε απλώς κλικ στο κουμπί Αρχική σελίδα στα αριστερά.

1.2.5.3. Εγκατάσταση του Jupyter-Lab#

Για να ξεκινήσετε να χρησιμοποιείτε την Python, υπάρχουν λίγες διαφορετικές επιλογές διαθέσιμες για σας. Θα μπορούσατε να χρησιμοποιήσετε ένα τερματικό και να ακολουθήσετε το εγχειρίδιο, αλλά αυτό μπορεί να είναι αρκετά δύσκολο εάν μόλις ξεκινάτε. Δεδομένου ότι αυτό το εγχειρίδιο σχεδιάστηκε στο JupyterLab, νομίζω ότι έχει περισσότερο νόημα να εγκαταστήσετε αυτό το λογισμικό. Προχωρήστε και κάντε κλικ Εγκατάσταση κάτω από το JupyterLab. Μόλις εγκατασταθεί, θα μπορείτε να το Εκτελέσετε και να ακολουθήσετε το εγχειρίδιο.

1.2.6. Συμπέρασμα#

Εάν σε κάποιο σημείο της διαδικασίας εγκατάστασης νιώσετε απογοήτευση, μην αποθαρρυνθείτε. Δεν είστε μόνοι και δεν είναι μια απλή ή εύκολη διαδικασία. Αυτοί που προγραμματίζουν χρόνια μπορούν να κάνουν αυτά τα βήματα να φαίνονται απλά, αλλά η αντιμετώπιση προβλημάτων μόνοι σας είναι προκλητική. Είναι σημαντικό να θυμάστε ότι ο προγραμματισμός γενικά είναι προκλητικός. Προσπαθήστε να κοιτάξετε τις προκλήσεις που προκύπτουν ως διασκεδαστικά προβλήματα που πρέπει να επιλυθούν. Προσπαθήστε να δείτε τα λάθη σας μέσα από ένα φακό Bob Ross και να τα δείτε ως ευτυχισμένα ατυχήματα.
